## 🛠️ 기술 스택

- **프레임워크**: Next.js 16.0.7 (App Router)
- **언어**: TypeScript 5
- **스타일링**: Tailwind CSS 3.4.4
- **UI 컴포넌트**: Radix UI, shadcn/ui
- **아이콘**: Lucide React, Font Awesome
- **상태 관리**: React Hooks
- **데이터베이스**: Supabase (준비 중)
- **인증**: NextAuth.js (준비 중)

## 📦 설치 및 실행

### 1. 저장소 클론

```bash
git clone <repository-url>
cd vibefolio-nextjs
```

### 2. 의존성 설치

```bash
npm install
```

### 3. 개발 서버 실행

```bash
npm run dev
```

브라우저에서 [http://localhost:4000](http://localhost:4000)을 열어 확인하세요.

### 4. 프로덕션 빌드

```bash
npm run build
npm start
```

## 🎯 사용 방법

### 프로젝트 등록하기

1. 메인 페이지 우측 하단의 **검은색 플로팅 버튼(+)** 클릭
2. 프로젝트 이미지 업로드
3. 제목, 설명, 카테고리 입력
4. **프로젝트 등록** 버튼 클릭
5. 메인 페이지에서 Masonry 그리드로 확인

### 카테고리 필터링

- Sticky Menu에서 **전체**, **AI**, **영상/모션그래픽** 선택
- 선택한 카테고리의 프로젝트만 표시

### 프로젝트 카드 호버

- 카드에 마우스를 올리면 어두운 오버레이와 함께 좋아요/조회수 표시
- 부드러운 애니메이션 효과

## 🎨 디자인 시스템

### 색상 팔레트

- **배경**: White (#FFFFFF), Gray-50 (#F9FAFB)
- **텍스트 Primary**: Gray-900 (#111827)
- **텍스트 Secondary**: Gray-600 (#4B5563)
- **구분선**: Gray-200 (#E5E7EB)
- **버튼**: Black (#000000)
- **호버**: Gray-800 (#1F2937)

### 타이포그래피

- **시스템 폰트**: -apple-system, BlinkMacSystemFont, Segoe UI, Roboto
- **제목**: 24-48px, Bold
- **본문**: 14-16px, Regular/Medium
- **캡션**: 12-14px, Regular

### 레이아웃

- **최대 너비**: 1280px (max-w-7xl)
- **그리드**: Masonry (1-4열, 반응형)
- **간격**: 24px
- **패딩**: 24-48px

### 컴포넌트 스타일

- **카드**: 흰색 배경, 1px 회색 테두리, 4px 라운드
- **버튼**: 검은색 배경, 4px 라운드, 호버 시 어두워짐
- **호버 효과**: 2px 위로 이동, 그림자 강화
- **오버레이**: 검은색 40% 투명도
